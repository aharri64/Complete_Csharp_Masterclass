# Decisions in C#

## If Statement

```cs
if(condition)
{
    //what to do if condition is met
}
```

```cs
if(condition){
    //what to do if condition is met
}
else
{
    // what to do if condition is not met
}
```

```cs
if(condition)
{
    //what to do if condition is met
}
else if (condition)
{
    // what to do if condition2 is met
}
else
{
    // what to do if condition is not met
}
```
